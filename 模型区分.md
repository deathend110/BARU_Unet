| 模型           | 年份         | 核心思想                        | Attention类型                  | 与NAFNet差异              |
| ------------ | ---------- | --------------------------- | ---------------------------- | ---------------------- |
| **NAFNet**   | ECCV 2022  | Activation-Free CNN         | Simplified Channel Attention | 基准模型                   |
| **KBNet**    | ICCV 2023  | Kernel Basis                | Kernel Basis Attention       | CNN Attention增强        |
| **MIRNetv2** | TPAMI 2022 | Multi-scale Feature Fusion  | Selective Kernel/Attention   | 多尺度恢复                  |
| **HINet**    | CVPRW 2021 | Half Instance Normalization | 几乎无Attention                 | 双阶段UNet                |
| **SCUNet**   | ICCV 2023  | Conv + Swin                 | Window Self-Attention        | Hybrid CNN-Transformer |

## 模型设置
为了论文和专利的可比性，建议所有模型统一：

| 项目                | 设置                          |
| ----------------- | --------------------------- |
| 输入                | 256×256 灰度图                 |
| Batch Size        | 16（不足则8）                    |
| Optimizer         | AdamW                       |
| Loss              | L1 + SSIM（后续可加入Charbonnier） |
| Epoch             | 300~500                     |
| Learning Rate     | 2×10⁻⁴（Cosine或StepLR）       |
| Data Augmentation | Flip + Rotate               |
